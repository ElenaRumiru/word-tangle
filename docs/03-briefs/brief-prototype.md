# Бриф: интерактивный веб-прототип

Исполнитель: prototype-engineer. Смоук: qa-playtester.
Deliverable: `deliverables/prototype/jumble-mobile.html` — ОДИН самодостаточный файл.

## Технические требования

- Vanilla HTML/CSS/JS, без сборки, без сети. Открылся двойным кликом — работает.
- Уровни: JSON из `deliverables/levels/levels.json` инлайном в `<script>`.
- Картинки: base64 data-URI инлайном (10 шт. × ~0.5–1 МБ — файл до ~15 МБ допустим;
  при желании сжать до webp/jpeg q70 скриптом перед встраиванием).
- Мобильный вьюпорт: макет под 390×844, `meta viewport`, тач-события + мышь (для десктоп-проверки),
  на десктопе — центрированная «рамка телефона».
- Прогресс в localStorage (пройденные уровни, валюта подсказок).
- Публикация: файл дополнительно можно выкатить как Artifact (шаринг ссылкой) — контент
  self-contained, CSP-совместим (никаких внешних запросов).

## Функциональный объём (по ux-concept.md фазы 1)

1. Экран Home: логотип-заголовок (свой, не трейдмарк Jumble — например «Word Tangle» / придумать),
   кнопка Play, выбор уровня (карта 1–10 с замками и галочками).
2. Игровой экран, фаза слов: скрэмбл-плитки (tap-to-place), слоты с кружками, Shuffle,
   кнопки подсказок, решённые слова сворачиваются, кружки улетают в буфер ответа.
3. Фаза ответа: карикатура (тизер → полное раскрытие), подпись-подводка, слоты ответа
   с разбивкой по словам и кавычками, тот же tap-to-place из букв буфера.
4. Победный экран: конфетти (CSS/canvas достаточно), полный ответ, кнопка «Дальше».
5. Джус-минимум: pop плитки, shake на ошибку, каскад на верное слово, похвалы-словечки.
6. Подсказки: shuffle бесплатно; открыть букву; открыть позицию; «объяснение шутки» на фазе ответа;
   счётчик монет (стартовый запас, расход по лестнице цен — цифры взять из balance-notes или задать
   разумные: 25/50/100).

## Читы (ОБЯЗАТЕЛЬНОЕ требование задания)

Чит-панель открывается кнопкой «CHEATS» в углу (всегда видима — ревьюер не должен искать) и содержит:
- **Unlock all levels** — снять замки со всех уровней.
- **Jump to level N** — мгновенный переход.
- **Solve current word** — решить текущее слово.
- **Solve level** — мгновенно решить весь уровень (слова + ответ).
- **Reveal answer** — показать ответ-сюрприз без завершения.
- **+1000 coins** — валюта подсказок.
- **Reset progress** — очистить localStorage.
- **Difficulty overlay** — показать P1–P8/score текущего уровня поверх UI (для оценки баланса).
Каждый чит подписан по-русски. Состояние «читы использованы» может помечаться в UI (звёздочка) — опционально.
