# Бриф: 10 уровней и модель сложности

Исполнитель: word-game-designer (+ ревью balance-economist).
Deliverables: `deliverables/levels/levels.json`, `deliverables/levels/levels-design.md`.

## Требование задания

10 уровней; сложность нарастает так, будто 10-й уровень — это **100-й уровень в игре**.
То есть уровни прототипа сэмплируют полную кривую 1–100: уровень k прототипа ≈ уровень
`1, 12, 23, 34, 45, 56, 67, 78, 89, 100` полной игры (равномерный шаг ~11). Балансовая таблица
(brief-balance-table) обязана давать те же значения в этих точках — единая модель, два артефакта.

## Параметры сложности (единая система с балансом)

| # | Параметр | Уровень 1 | Уровень 10 (=100) |
|---|---|---|---|
| P1 | Число слов | 1–2 | 5 |
| P2 | Длина слов | 4 | 6–7 |
| P3 | Частотность (Zipf-банд словаря) | топ-1000 слов | редкая лексика |
| P4 | Скрэмбл-скор (см. ниже) | лёгкий (сохранены биграммы/первая буква) | максимальная дезориентация |
| P5 | Анаграммы-ловушки | нет | набор букв с правдоподобной ложной анаграммой |
| P6 | Букв в ответе / слов в ответе | 4–5 / 1 | 12+ / 3 |
| P7 | Тип каламбура | прямое слово | омофон / двухслойная игра слов |
| P8 | Прозрачность карикатуры | почти показывает ответ | косвенный намёк |

**Скрэмбл-скор** (для levels-design и генератора): доля разрушенных биграмм исходного слова +
штраф, если первая буква на месте (легче) + бонус, если скрэмбл произносим как псевдослово
(«залипает»). Формализовать в levels-design.md (простая формула 0–1 достаточна).

Кривая: восходящая пила — общий тренд вверх, локальные передышки (уровень после спайка чуть легче).
На 10 уровнях: спайки на 4, 7, 10; передышки на 5, 8.

## Формат данных уровня (levels.json)

```json
{
  "schema": "jumble-level.v1",
  "levels": [
    {
      "id": 1,
      "virtualLevel": 1,
      "words": [
        { "answer": "SLOVO", "scramble": "VOSOL", "circles": [0, 2] }
      ],
      "surprise": {
        "caption": "Подпись-подводка карикатуры…",
        "answer": ["WADE", "A", "MINUTE"],
        "quoted": [0],
        "explanation": "Текст нарративной подсказки (объяснение шутки)"
      },
      "cartoon": { "file": "level-01.png", "prompt": "…промпт для art-director…" },
      "difficulty": { "score": 1.0, "P1": 2, "P2": 4, "P3": 1, "P4": 0.2, "P5": 0, "P6": 5, "P7": "direct", "P8": 3 }
    }
  ]
}
```

`circles` — индексы букв слова-ответа, попадающих в буфер сюрприза. Мультибуквенные правила:
мультисет букв всех кружков == мультисет букв ответа (проверяется скриптом).

## Обязательная валидация (перед сдачей фазы)

Написать и прогнать скрипт `deliverables/levels/validate_levels.py`:
1. Скрэмбл — перестановка своего answer.
2. Скрэмбл != answer и (для языка контента) не является сам словарным словом.
3. Мультисет букв кружков == мультисет букв ответа-сюрприза.
4. Answer каждого слова — словарное слово; в идеале проверка единственности анаграммы
   (по частотному словарю; если у набора букв есть вторая ходовая анаграмма — она либо
   осознанная ловушка P5 на поздних уровнях, либо слово заменяется).
5. Монотонность difficulty.score по уровням с допуском на передышки.

## Контент

- Язык — по решению пользователя (открытый вопрос №1 мастер-плана).
- Если EN: сырьё — 97 пазлов журнала (`01-research/jumble-solutions-raw.txt`); ранние уровни можно
  собирать из усечённых журнальных пазлов (меньше слов, легче скрэмблы), поздние — авторские.
- Если RU: контент авторский; каламбуры строить на омофонии/идиомах («сел в лужу», «не в своей тарелке» —
  идиомы с буквальной визуализацией отлично конвертируются в карикатуры). Слова проверять по
  частотному списку русского (НКРЯ / Шаров) — банды P3.
- Тон шуток: семейный, тёплый, без политики/чернухи — как в оригинале.
- В levels-design.md для каждого уровня: таблица P1–P8, целевое время решения, чем создан спайк/передышка.

## Чек-лист сдачи

- [ ] 10 уровней в levels.json валидны по схеме и скрипту.
- [ ] Кривая 1→10 документирована и согласована с точками 1→100 полной кривой.
- [ ] У каждого уровня есть промпт карикатуры (по стайл-гайду art-брифа).
- [ ] Ответы-сюрпризы — действительно каламбуры с нарастающей хитростью, а не просто слова.
- [ ] Уровень 1 проходится за <30 сек без подсказок; уровень 10 заставляет думать 3+ минуты.
